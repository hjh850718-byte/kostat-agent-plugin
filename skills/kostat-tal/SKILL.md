# kostat-tal (판단 기준)

## 개요
KOSTAT 업무에서 자주 사용되는 암묵적 판단 규칙을 명문화한 참조 자료.

## 주요 판단 기준

### MFG SITE 매핑
| 조건 | 기본값 |
|------|--------|
| 기존 Kostat PN 존재 | 이전 행 MFG SITE 값 따름 |
| 신규 Kostat PN | 'KR' 기본값 |

### 고객사별 특이사항
- **Skyworks**: PO 형식 일정, Ship To Code 주의
- **Micron**: 복수 라인 PO 빈번, 라인별 분리 필수
- **Qorvo**: Remarks 형식 엄격

### 우선순위 규칙
- 데이터 충돌 시: PO PDF 원문 > Excel 기존 값 > 추정
- 중복 PO# 발생 시: 자동 덮어쓰기 금지, 사용자 확인 필수
- 날짜 표준: YYYY-MM-DD 통일

## 검증 체크리스트 (Loop 방식)
- [ ] 조회한 판단 기준이 현재 작업 상황에 정확히 적용되는가
- [ ] 고객사별 특이사항 반영 여부 확인
- [ ] MFG SITE 매핑 규칙 준수

## 검증 루프 절차
1. **체크리스트 1회 실행** — 위 항목 순차 점검
2. **실패 항목 발견 시** → 기준 재조회 → 재적용 → 다시 1단계로 (Loop)
3. **전체 통과 시** → 다음 작업 진행

### Loop 규칙
- 실패가 0이 될 때까지 반복 (최대 3회)
- 재검증 시 이전과 다른 각도에서 검증 (동일 실수 반복 방지)
- 3회 초과 실패 → 사용자에게 상황 보고 후 에스컬레이션
