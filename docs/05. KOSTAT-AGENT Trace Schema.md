## 4. Notion DB 생성 방법

현재 MCP `create-a-data-source` API가 지원되지 않아 Notion UI에서 직접 생성해야 합니다.

```
1. 🏢 KOSTAT 해외영업 운영 대시보드 페이지 열기
2. 페이지 빈 칸에 '/database' 입력 → 'Table' 선택
3. 데이터베이스 제목: "📊 KOSTAT-AGENT Trace Log"
4. 위 속성 정의표대로 컬럼 추가
   - Title(기본) → "작업명"으로 이름 변경
   - '+ Add a property'로 나머지 속성 추가
5. Select 옵션 등록:
   - 스킬명: 6개 옵션 (위 표 참고)
   - 성공여부: 3개 옵션
```

## 5. Hook 연동 (Phase 3 예정)

PostToolUse Hook이 이 DB에 자동 기록:

```json
{
  "PostToolUse": [
    {
      "matcher": "Write|Edit",
      "hooks": [
        {
          "type": "command",
          "command": "python kostat-trace-logger.py",
          "timeout": 10
        }
      ]
    }
  ]
}
```

`kostat-trace-logger.py` 동작:
1. Write/Edit 도구 호출 감지
2. 파일 경로, 작업 내용 파싱
3. Notion API로 Trace DB에 레코드 추가
4. 실패 시 로컬 fallback 로그 저장
