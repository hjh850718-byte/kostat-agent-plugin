# Make.com + Claude (Anthropic API) 자동화 구현 가이드

> n8n 대신 Make.com (구 Integromat) 기준으로 재작성. Make.com은 클라우드 SaaS라 설치 불필요.

---

## 1. 사전 준비

### 필수 환경
| 항목 | 내용 |
|------|------|
| Make.com | 가입 https://www.make.com (무료 티어: 1,000 ops/월) |
| AI 모델 | Anthropic API (claude-sonnet-4-6 권장) |
| 검색 API | Brave Search API 또는 SerpAPI |
| 추가 모듈 | Slack, Gmail, Notion, Telegram (Make.com 기본 탑재) |

### API 키 사전 발급
- **Anthropic API Key**: https://console.anthropic.com 에서 발급
- **Brave Search API**: https://api.search.brave.com 에서 발급
- **필요시**: Slack Bot Token, Telegram Bot Token, Notion API Key

## 2. Make.com 가입 및 기본 설정

### 가입
1. https://www.make.com 접속 → **Get started free**
2. 이메일 가입 또는 Google/GitHub OAuth
3. 대시보드에서 **Create a new scenario** 클릭

### Make.com 핵심 개념
| 용어 | 설명 |
|------|------|
| **Scenario** | 자동화 단위 (n8n의 워크플로에 대응) |
| **Module** | 각 처리 단계 (n8n의 노드에 대응) |
| **Bundle** | 모듈 간 주고받는 데이터 단위 (JSON) |
| **Operation** | 실행 1회 (시나리오 1회 실행 = 1 op) |
| **Data Structure** | JSON 스키마 정의 (출력 형식 고정용) |
| **Webhook** | 외부에서 시나리오 트리거 |

> Make.com은 클라우드 SaaS이므로 Docker/로컬 설치가 필요 없다. 모든 것이 브라우저에서 동작한다.

## 3. Claude (Anthropic) 모듈 연결

Make.com은 **Claude 전용 모듈이 없으므로** HTTP 모듈로 Anthropic API를 직접 호출한다.

### HTTP 모듈 → Anthropic API 설정

**Module**: `HTTP > Make a request`

**설정값:**
```
URL: https://api.anthropic.com/v1/messages
Method: POST
Headers:
  Content-Type: application/json
  x-api-key: {{your_anthropic_api_key}}
  anthropic-version: 2023-06-01
Body (JSON):
{
  "model": "claude-sonnet-4-20250514",
  "max_tokens": 4096,
  "messages": [
    {
      "role": "user",
      "content": "{{prompt_text}}"
    }
  ]
}
```

> **팁**: API Key는 Make.com의 **Variables** 기능에 저장해 재사용.

### Claude 응답에서 텍스트 추출

응답 JSON 구조:
```json
{
  "content": [
    { "type": "text", "text": "여기가 응답 내용" }
  ]
}
```

**추출 매핑**: `{{data.content[0].text}}`

## 4. 외부 API 검색 (MCP 대체)

Make.com은 MCP 프로토콜을 직접 지원하지 않는다. 대신 **HTTP 모듈**로 외부 검색 API를 직접 호출한다.

### Brave Search API 예시

**Module**: `HTTP > Make a request`

```
URL: https://api.search.brave.com/res/v1/web/search?q={{keyword}}&count=5
Method: GET
Headers:
  Accept: application/json
  Accept-Encoding: gzip
  X-Subscription-Token: {{brave_api_key}}
```

### Google Custom Search API 예시

```
URL: https://www.googleapis.com/customsearch/v1?key={{API_KEY}}&cx={{CX}}&q={{keyword}}
Method: GET
```

## 5. 시나리오 구성 (모듈 순서)

### 기본 패턴: 뉴스 수집 → Claude 요약 → Telegram 전송

```
Schedule (매일 오전 8:00)
    ↓
HTTP (Brave Search API) — "Skyworks 최신 뉴스 검색"
    ↓
HTTP (Brave Search API) — "Micron 최신 뉴스 검색"
    ↓
HTTP (Brave Search API) — "Infineon 최신 뉴스 검색"
    ↓
Array Aggregator — 3개 검색 결과 합치기
    ↓
HTTP (Anthropic API) — Claude 요약 프롬프트
    ↓
Text Parser — Claude 응답에서 JSON 추출
    ↓
Telegram — 요약 메시지 전송
```

### 각 모듈 핵심 설정

**Schedule**
- Trigger: `Repeat every day at 08:00`
- Day: `Monday-Friday` (주말 제외)

**HTTP (Brave Search)**
- URL에 `{{keyword}}` 동적 매핑
- 응답에서 `web.results[]` 배열 추출

**Array Aggregator**
- Source Module: HTTP (Brave Search) 3개 중 마지막
- **모든 검색 결과를 하나의 배열로 병합**

**HTTP (Anthropic API)** — 요약 프롬프트:
```json
{
  "model": "claude-sonnet-4-20250514",
  "max_tokens": 2048,
  "messages": [{
    "role": "user",
    "content": "아래 검색 결과를 요약해줘. 각 항목별로 핵심 내용을 2-3문장으로:\n\n{{array_aggregated}}"
  }]
}
```

**Telegram**
- Bot Token: Make.com Variables에 저장
- Message: `{{claude_response}}`

## 6. 프롬프트 작성 원칙

Make.com에서 Claude API를 호출할 때의 프롬프트 원칙:

### ① 역할 정의
```
당신은 KOSTAT 해외영업부의 분석 어시스턴트입니다.
반드시 한국어로 간결하게 응답합니다.
```

### ② 실행 지침 (단계 명시)
```
1. 제공된 데이터를 분석하세요.
2. customer별로 그룹화하여 요약하세요.
3. 이상 징후가 있으면 별도 표시하세요.
4. JSON 형식으로만 출력하세요.
```

### ③ 품질/제약 조건
```
- 검증된 정보만 포함할 것
- 추측하지 말 것
- 마크다운 사용 금지
```

### ④ 출력 형식 고정 — Data Structure 정의

Make.com의 **Data Structure** 기능으로 JSON 출력 형식을 강제:

```
Data Structure Name: ClaudeResponse
Definition:
{
  "summary": "String",
  "items": [
    {
      "title": "String",
      "key_point": "String",
      "url": "String"
    }
  ]
}
```

## 10. 트러블슈팅

**Claude API 401 오류**
→ `x-api-key` 헤더에 Variables에서 올바르게 매핑되었는지 확인

**검색 결과 없음**
→ Brave Search API 할당량 확인 (무료 티어: 2,000회/월)

**JSON 파싱 실패**
→ Claude 프롬프트에 `"JSON 형식으로만 출력. 설명이나 부연 없음."` 추가
→ Text Parser로 Claude 응답에서 `{` ~ `}` 블록 추출

**Telegram 메시지 안 옴**
→ Bot Token 확인. Make.com Telegram 모듈에서 Bot을 먼저 `Create a connection`

**시나리오 실행 시간 초과**
→ Claude `max_tokens`를 줄이거나, 검색 결과 수 제한

**Make.com 무료 티어 제한**
→ 1,000 ops/월. 하루 33회 = 1시간 간격 1개 시나리오 정도 가능.
