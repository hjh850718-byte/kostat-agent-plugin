# kostat-po-update Eval 테스트 케이스 v1.0

> 대상: `kostat-po-update` 스킬 (PO PDF → Excel 자동 입력)  
> 기준 파일: `미국오더 20260410-기존.xlsx` — 미국오더 시트 (2419행 기준, 42컬럼)  
> 검증 환경: Claude Code PowerShell + openpyxl  
> 작성일: 2026-06-04

---

## 테스트 실행 절차

1. 각 케이스별 입력 PDF를 지정 경로에 준비
2. 스킬 실행 (또는 동일 로직을 Python으로 호출)
3. 기대 결과와 실제 결과 비교
4. 합격/불합격 판정 + 실패 시 원인 기록
5. 결과를 `evals/po-update-eval-log.md`에 누적 기록

def hash_rows(ws, max_row):
    """기존 행의 해시를 저장해 무결성 검증에 사용"""
    hashes = []
    for r in range(1, max_row+1):
        row_vals = tuple(str(ws.cell(r, c).value) for c in range(1, ws.max_column+1))
        hashes.append(hashlib.md5(str(row_vals).encode()).hexdigest())
